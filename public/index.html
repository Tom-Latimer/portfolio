PakuPaku

The PakuPaku game is designed as a web-based 1D grid arcade game, featuring a simple yet engaging gameplay where players control Pakupaku, a character that navigates a linear track to consume pellets and fruits while evading ghosts. The game is implemented using HTML, CSS for styling, and JavaScript for game mechanics, ensuring a responsive and intuitive user experience. The game board and elements are styled with a dark color palette to enhance the retro arcade aesthetic, and the controls are limited to left and right arrow keys for movement, making it accessible and easy to master. The game features a dynamic scoreboard that updates in real time, showing current and high scores, and provides a 'Game Over' screen with an option to restart the game upon collision with ghosts.

PakuPaku-PHP

The PakuPaku-PHP game is a simplified, linear version of Pac-Man, implemented using PHP for backend logic and JavaScript for frontend interaction. The game consists of several PHP components: GameEntity (an abstract class for movable entities like Pac-Man and ghosts), Paku (representing Pac-Man), and Ghost (representing enemy ghosts). The core game logic is managed by the PakuPakuGame class, which handles the game board, entity positions, and game status. The PakuPakuEngine class serves as the game engine, facilitating the interaction between the frontend and the game logic, managing game state updates, direction changes, and game resets. The frontend uses JavaScript and jQuery to handle user inputs, update the game display, and manage the game loop.

Mealer Application

In this project, we created a Mealer application, which allows customers to purchase meals from home cooks. These home cooks prepare the meal at their own houses and list the meals for sale on the Mealer app. In our project, we implemented this concept using Android Studio to create the UI and used Java for the backend work. Additionally, we used Firebase Authentication to implement the login process and utilized the Firestore Database to store the remainder of the data, including meals, reviews, permissions, meal requests, and different user permissions.
