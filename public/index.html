Mealer Application

In this project, we created a Mealer application, which allows customers to purchase meals from home cooks. Additionally, we used Firebase Authentication to implement the login process and utilized the Firestore Database to store the remainder of the data, including meals, reviews, permissions, meal requests, and different user permissions.
